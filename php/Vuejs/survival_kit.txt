Install Node JS:
https://nodejs.org/en/download/

Install Vue JS:
https://vuejs.org/v2/guide/installation.html (Using Vue CLI)
sudo npm install -g @vue/cli

Create a new app project:
vue create my_first_vuejs_proj

Run the serve:
cd my_first_vuejs_proj/
npm run serve
