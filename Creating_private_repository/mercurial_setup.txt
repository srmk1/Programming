Download Mercurial: https://www.mercurial-scm.org/

On your computer:
1. cd to directory where you have source code parent directory
2. hg init
3. Add all python files under this directory
   hg add python_fundamentals/*.py
4. Check status
   hg status
5. Add username
    hg config --edit    ---> Edit [username] = srmk
6. Commit the scripts
    hg commit
7. Check the branch
    hg tip


For Others to Clone setup server
1. Create a config file
    vi /Users/srmk/hgweb.config   ==> Add following lines at the end of hgweb.config
          [paths]
          python_fundamentals = /Users/srmk/PycharmProjects/PluralSightProject
2. Update .hgrc
    cd      ==> Go to home directory
    vi .hgrc  ==> add following lines at the end of .hgrc
          #WEB
          [web]
          allow_push = *
          push_ssl = false
3. Serve the repository
   hg serve -a 10.232.28.16 -p 8097 --web-conf /Users/srmk/hgweb.config -d
